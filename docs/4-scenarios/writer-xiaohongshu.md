# 小红书文案创作

小红书是当下最具商业价值的平台之一。但很多人卡在"写什么"和"怎么写"。

本章将教你一套 **AI 驱动的小红书文案工作流**，让你：
- ✅ 不再为选题发愁
- ✅ 10 分钟产出 1 篇高质量文案
- ✅ 形成可复用的内容模板

## 为什么选小红书

| 优势 | 说明 |
|------|------|
| 流量红利 | 算法友好，新人也有曝光机会 |
| 变现路径清晰 | 接广告、带货、引流私域 |
| 内容门槛低 | 图文为主，不需要专业设备 |
| 用户质量高 | 一二线城市年轻女性为主，消费力强 |

## 核心工作流

```
选题 → 标题 → 正文 → 排版 → 发布
  ↑                              ↓
  └──────── 数据复盘 ← 互动维护 ┘
```

## 第一步：用 AI 批量选题

### 提示词模板

```
你是我的选题助手。请基于【美妆/穿搭/职场】领域，
生成 20 个小红书选题，要求：
1. 针对 25-35 岁都市女性
2. 有实用价值或情绪价值
3. 容易获得高互动（收藏/评论）
4. 避开过于竞争的大词

请按以下格式输出：
- 选题名称
- 目标痛点
- 预期互动类型（收藏/评论/点赞）
```

### 实操示例

用 Claude 运行上述提示词，你会得到类似这样的结果：

| 选题 | 痛点 | 互动类型 |
|------|------|---------|
| 早八人 5 分钟伪素颜妆 | 没时间化妆 | 收藏 |
| 被同事问了 800 次的香水 | 社交认可 | 评论 |
| 月薪 5k 到 5w 的穿搭变化 | 成长焦虑 | 点赞+评论 |

## 第二步：生成爆款标题

### 标题公式

小红书爆款标题 = **数字** + **痛点/利益** + **情绪词**

### 提示词模板

```
基于选题【{选题}】，生成 10 个小红书标题。

要求：
1. 包含具体数字（3 个、5 步、10 元）
2. 制造紧迫感或好奇心
3. 口语化，像闺蜜聊天
4. 适当使用 emoji

避免：
- 标题党，与内容不符
- 过于官方的语气
- 敏感词（最、第一、绝对）
```

### 好的标题示例

❌ 错误的标题：
> 如何化好妆

✅ 正确的标题：
> 手残党进！3 步画出伪素颜妆，被同事追着问 😭

## 第三步：撰写正文

### 正文结构

```
开头（抓眼球）→ 痛点（共鸣）→ 解决方案（干货）→ 结尾（互动）
```

### 提示词模板

```
请写一篇小红书文案。

选题：{选题}
标题：{标题}

要求：
1. 开头 2 句话抓住注意力
2. 描述一个具体痛点场景
3. 给出 3-5 个可执行的建议
4. 每段用 emoji 分隔，增加可读性
5. 结尾引导评论互动
6. 全文 300-500 字

风格：
- 像闺蜜分享经验
- 真实、接地气
- 适当自黑
```

## 第四步：排版优化

### 排版原则

1. **短段落** - 每段不超过 2 行
2. **多用 emoji** - 视觉分隔，增加亲和力
3. **重点突出** - 核心信息用 emoji 标注
4. **留白充足** - 阅读不累

### 排版检查清单

- [ ] 开头有 emoji 或数字
- [ ] 段落间有空行
- [ ] 关键信息用 **加粗** 或 emoji 标注
- [ ] 结尾有互动引导
- [ ] 全文不超过 5 屏（手机端）

## 第五步：发布与复盘

### 发布时机

| 时间段 | 适合内容 |
|--------|---------|
| 早 7-9 点 | 职场、学习类 |
| 午 12-14 点 | 娱乐、生活类 |
| 晚 20-22 点 | 情感、消费类 |

### 数据复盘表

| 指标 | 说明 | 优化方向 |
|------|------|---------|
| 曝光量 | 有多少人看到 | 标题/封面 |
| 点击率 | 看到的人有多少点击 | 封面/标题 |
| 互动率 | 点赞+收藏+评论/曝光 | 内容质量 |
| 收藏率 | 收藏/曝光 | 干货程度 |

## 常见问题

**Q：AI 生成的文案会不会太千篇一律？**

A：会！你需要：
1. 在提示词中加入个人经历
2. 生成后手动修改细节
3. 加入自己的口头禅或风格

**Q：需要多久才能起号成功？**

A：坚持日更 1 个月，数据会逐渐起来。关键是持续优化，不是盲目发。

**Q：什么领域最适合新手？**

A：选你已经在做的、或很想学的领域。真实感最重要。

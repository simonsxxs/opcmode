## 学习建议

1. **先打好基础** - 确保基础功能已熟练使用
2. **按需学习** - 不需要全部看完，用到什么学什么
3. **多看源码** - 很多高级用法都在源码里
4. **勇于尝试** - 配置坏了可以恢复，多试错才能进步

# 开始学习

欢迎来到 **AI+一人公司入门课** 的第一章。

本章将带你从零开始，在 7 天内完成：**定位** → **产品** → **验证** 的完整闭环。

## 你将学到什么

- ✅ 如何找到适合自己的定位
- ✅ 如何做出最小可售卖的服务
- ✅ 如何完成第一单成交验证
- ✅ 如何搭建高效的 AI 工作环境

## 学习建议

::: tip 💡 提示
不要跳过任何步骤，按顺序完成。每一步都是下一步的基础。
:::

### 每日学习计划

| 天数 | 任务 | 目标 |
|------|------|------|
| Day 1 | 看介绍 + 安装环境 | 能打开 Claude Code |
| Day 2 | 配置网络 + 连接 AI | 能正常使用 AI 对话 |
| Day 3-4 | 探索各模型特点 | 找到适合自己的组合 |
| Day 5-7 | 实践练习 | 完成第一个小任务 |

## 预期收获

完成本章后，你将：

1. **拥有一个可用的 AI 编程助手环境**
2. **了解如何选择和配置不同的 AI 模型**
3. **掌握基本的网络故障排查能力**
4. **为后续的场景实战打下坚实基础**

# 课程介绍

欢迎来到 **AI+一人公司入门课**。这是一门专为小白设计的实战课程，帮助你从零开始，用 AI 工具搭建属于自己的一人公司业务。

## 为什么是"AI+一人公司"

传统的一人公司需要：
- ✅ 技术能力
- ✅ 设计能力
- ✅ 营销能力
- ✅ 客户服务能力

**普通人很难同时具备这些能力。**

但有了 AI：
- AI 帮你写代码 → 技术门槛降低
- AI 帮你设计 → 审美门槛降低
- AI 帮你写文案 → 营销门槛降低
- AI 帮你做客服 → 服务门槛降低

**一个人 + AI = 一个团队**

## 课程目标

本课程的目标是：

> **在 30 天内，跑通从定位到成交的完整闭环，完成你的第一单。**

不是教你成为专家，而是教你：
1. **快速起步**：别犹豫，先动起来
2. **小步快跑**：用最小成本验证想法
3. **持续迭代**：根据反馈不断优化

## 适合谁学

✅ **完全新手** - 没有技术背景，但有强烈的学习意愿
✅ **想转型的人** - 厌倦 996，想尝试自由职业
✅ **有主业的人** - 想发展副业，增加收入来源
✅ **小创业者** - 已有想法，但不知道如何落地

**不适合：**
❌ 追求速成、不想付出努力的人
❌ 期待一夜暴富的人
❌ 完全不愿意学习新工具的人

## 课程结构

### 第一章：7天起步
- 明确定位
- 搭建环境
- 做出第一个产品雏形

### 第二章：日常使用
- 掌握 Claude Code 的核心功能
- 建立高效的工作流

### 第三章：工作流模板
- 项目规划
- Agent 使用
- 自动化脚本

### 第四章：场景实战
- AI 写作工作流
- 程序员日常使用
- 办公自动化

### 第五章：进阶手册
- MCP 系统
- Hooks 机制
- 自定义工具开发

## 学习路径建议

根据你的背景，选择适合的路径：

### 路径 A：完全新手（推荐）
```
第一章 → 第二章 → 第三章 → 第四章
```
预计时间：2-3 周

### 路径 B：有技术基础
```
第一章（跳过安装部分） → 第四章 → 第五章
```
预计时间：1-2 周

### 路径 C：急需变现
```
第一章 → 第四章（选择最相关的场景） → 实战
```
预计时间：1 周内开始尝试
